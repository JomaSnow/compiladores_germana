## Objetivo

O objetivo deste projeto foi desenvolver um compilador para a linguagem C implementando analisadores léxicos, sintáticos, semânticos e geradores de código.

## Instalação

Install flex:
    
- Ubuntu:

        sudo apt install flex 

- Mac:
        
        brew install flex

Install bison:

- Ubuntu:
        
        sudo apt install bison
- Mac:
        
        brew install bison

## Execução

    make run
